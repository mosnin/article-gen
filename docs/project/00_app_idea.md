# App Idea — ArticleGen

## One-Liner

AI-powered article generation and multi-platform publishing tool for content creators, bloggers, and SEO professionals.

## Problem

Writing high-quality, SEO-optimized articles is time-consuming. Publishing to multiple platforms (WordPress, Medium, Dev.to, Ghost, Shopify) requires separate workflows. SEO keyword research, meta tags, and content clustering are complex skills most writers lack.

## Solution

ArticleGen automates the full content pipeline: research → outline → article → images → publish. Users provide a topic and keyword; the AI handles everything else, outputting publish-ready content with SEO metadata, schema markup, and AI-generated images.

## Core Experience

1. User signs up and completes onboarding (domain, site name, platform connections)
2. User enters a topic + focus keyword
3. AI generates research, outline, full article, metadata, and images in sequence
4. User reviews the result in a side-by-side preview
5. User publishes to connected platforms in one click
6. User tracks published/unpublished articles in their dashboard

## Generation Modes

- **Single** — one article, immediate generation
- **Batch** — queue of 10–50 articles, processed in sequence
- **Cluster** — pillar page + cluster articles with automatic interlinking

## Target Users

- Solo bloggers and content creators
- SEO professionals managing multiple sites
- Small agencies producing content at scale
- Developers and technical writers on platforms like Dev.to

## First Value Event

User sees their first complete, publish-ready article with metadata, images, and schema — within 60 seconds of submitting a topic.
